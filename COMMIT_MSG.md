## Описание
Реализован автокликер с графическим интерфейсом для автоматизации кликов на основе распознавания изображений.

## Ключевые возможности
- GUI на Tkinter с интуитивным управлением
- Сохранение изображений-образцов с выделением области
- Поиск изображений на экране (OpenCV matchTemplate)
- Настройка точности поиска (0.1–1.0) и задержки кликов (0.1–5.0с)
- Логирование событий с цветовой индикацией
- Фоновая работа в отдельном потоке

## Структура проекта
- `main.py` — GUI и точка входа
- `clicker.py` — логика кликера
- `screenshot.py` — модуль скриншотов
- `run.sh` — скрипт запуска с venv
- `clicker_images/` — папка для образцов

## Зависимости
pyautogui, Pillow, keyboard, opencv-python, numpy

## Требования
- Python 3.8+
- Linux (X11)
