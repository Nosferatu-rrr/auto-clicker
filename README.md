# Auto Clicker

Универсальный автокликер с распознаванием изображений для Linux.

## Описание

Простой и эффективный инструмент для автоматизации кликов на основе компьютерного зрения. Позволяет сохранять изображения-образцы, искать их на экране и выполнять клики при совпадении.

## Возможности

- Графический интерфейс на Tkinter
- Сохранение изображений-образцов с интерактивным выделением области
- Поиск изображений на экране с настраиваемой точностью
- Настройка задержки между кликами
- Логирование событий
- Работа в фоновом режиме

## Установка

```bash
# Установка зависимостей
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Или используйте скрипт запуска:
```bash
./run.sh
```

## Использование

1. **Сохранить образец:** Нажмите кнопку "📸 Скриншот", выделите область, задайте имя файла
2. **Запустить кликер:** Нажмите кнопку "🔴 Включить"
3. **Остановить:** Нажмите кнопку "⏹️ Стоп"

## Настройки

- **Точность поиска:** 0.1–1.0 (по умолчанию 0.8)
- **Задержка между кликами:** 0.1–5.0 сек (по умолчанию 0.5)
- **Папка с образцами:** `clicker_images/`

## Структура проекта

```
auto_clicker/
├── main.py           # GUI и точка входа

├── README.md         # Документация

```
